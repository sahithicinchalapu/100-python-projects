# Command line number guessing game (with difficulty modes)
## THis game is built in PYTHON 
---
## 🚀 Features

- 🎮 Three difficulty levels:
  - Easy (1–10)
  - Medium (1–50)
  - Hard (1–100)
- 🔢 Random number generation using Python's `random` module
- 📊 Attempt tracking system
- 💬 Real-time feedback:
  - Too High
  - Too Low
  - Correct Guess
- 🔁 Replay option after each game
- ⚠️ Handles invalid inputs gracefully
---
## 🧠 How It Works

1. User selects a difficulty level.
2. A random number is generated within the selected range.
3. The user keeps guessing until:
   - They guess correctly ✅
4. The game displays the number of attempts taken.
5. User can choose to play again.
